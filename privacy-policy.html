Privacy Policy for golden-record

Effective Date: July 15, 2026

This application is a personal, non-commercial automation tool designed to interact exclusively with the Pinterest API for the sole purpose of backing up and downloading images from the creator's personal Pinterest boards.

1. Data Collection

This application does not collect, store, or share any personal data, user credentials, or board information. All API interactions happen locally on the user's machine.

2. Data Usage

Any data accessed via the Pinterest API is used strictly to facilitate the local download of images and is not transmitted to any third parties.

3. Third-Party Services

This application uses the official Pinterest API. Your use of this tool is subject to Pinterest's standard Terms of Service and Privacy Policy.
